# drasticstatic — Default Community Health Files

This is the [`drasticstatic/.github`](https://github.com/drasticstatic/.github) repository — GitHub's special fallback location for community health files.

Any `drasticstatic` repository that does **not** have its own `SECURITY.md` or `CONTRIBUTING.md` automatically inherits the versions here.

---

## What's Here

| File | Purpose |
|------|---------|
| [`SECURITY.md`](SECURITY.md) | Vulnerability reporting policy — preferred channel, safe harbor, recognition |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | General contribution guidelines — fork, branch, PR |

---

## For Security Researchers

To report a vulnerability in any `drasticstatic` project, please use **[GitHub Private Vulnerability Reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)** on the affected repository's Security tab.

Confirmed reporters are credited in the advisory and listed in [CONTRIBUTORS.md](https://github.com/drasticstatic/drasticstatic/blob/main/CONTRIBUTORS.md).

## For Builders

Full repo scaffolding templates (`.gitignore`, `.augmentignore`, `CLAUDE.md`, sync workflows, branch protection, deploy scripts) live in [`drasticstatic/my-template`](https://github.com/drasticstatic/my-template).
